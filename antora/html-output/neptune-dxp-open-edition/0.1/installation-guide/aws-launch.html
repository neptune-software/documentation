Launch the AWS instance

Prerequisites

  • You created an instance on the AWS console.

Procedure

  1. Open your newly created instance.

  2. Choose one of the available Amazon Machine Images.

  3. Choose an instance type.

  4. Click Review.

  5. Click Launch.

    You need the key pair you created for that.

  6. Open the instance. You can check for the details of your instance here.

  7. Click Connect.

    The window Connect to Your Instance opens. There, you can configure the connection to the server.

Next steps

Now that you have launched the AWS instance, you must connect it to your Neptune DXP - Open Edition.
