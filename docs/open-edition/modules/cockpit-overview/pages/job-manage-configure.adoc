= Configure a job

In this topic, you learn how to configure a job and define who receives error notifications if the job fails.

== Prerequisites
* There are no prerequisites.

== Procedure

. In the Cockpit, navigate to *Tools* and click _Job Manage_.
. Click *Add* to add a new job.
+
*Result*: The Job Manage configuration window opens.
+
include::ROOT:partial$name-description-package.adoc[tags=name;description]
include::ROOT:partial$name-description-package.adoc[tags=package]
. If you want to start the job after you crated it, activate *Active*.
. In *Error Notification*, add email addresses that you want to notify if the job fails.
. Click *Save*.

== Results

* You have added an empty job.
* _Job Manage_ adds an ID and saves change dates.
