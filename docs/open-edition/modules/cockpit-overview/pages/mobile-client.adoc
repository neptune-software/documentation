_Mobile Client_ uses Apache Cordova plugins to create mobile applications.

Apache Cordova provides a way to develop mobile applications using standard web technologies: HTML5, CSS3 and JavaScript. Learn more: https://cordova.apache.org/[Apache Cordova website].
